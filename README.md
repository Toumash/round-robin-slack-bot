# round-robin-slack-bot
Manage people on duty from #alerts channel

| Command         	| Options         	| Description                                                                                                               	|
|-----------------	|-----------------	|---------------------------------------------------------------------------------------------------------------------------	|
| /alerts-today   	| [skip]          	| Shows today's user on the watch. If you pass skip parameter it will skip the person                                       	|
| /alerts-options 	| [list-of-names] 	| If you pass a list of users like `/alerts-options tomasz przemek` it will change the options and reset to the first entry 	|
